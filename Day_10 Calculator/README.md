
# Calculator Program

A calculator app which operates 4 basic add, subtract, multiply & divide operations.
This program is a good practice for Dictionaries & functions with outputs in Python.
```bash
EX:
Inputs:
  num1: 100
  num2: 20
  operation: /
Output: 
  5.0
```
